# 版本发布说明

**版本**: v2.0.0-beta.5
**日期**: 2026-04-29

---

## 新增功能

### 多语言支持
- 全新 **国际化(i18n)** 框架，基于 `react-i18next`
- 支持 **简体中文 / 繁体中文 / English / 日本語** 四种语言切换
- 语言设置即时生效，重启后保持
- 已完成翻译的核心页面：侧边栏、开始页、设置页(通用设置)、标题栏菜单

---

## 改进

### 后端与 API
- **日志配置持久化**: 日志路径、级别、轮转等设置现在保存到 `ai_config.json`，重启后依然生效
- **数据界面同步**: 开始页「最近卷轴」「待复习队列」「最近笔记」现在会监听卡片/笔记变更事件并自动刷新

---

## Bug 修复
- **数据界面同步**: 修复开始页「最近卷轴」「待复习队列」「最近笔记」不随卡片/笔记更新而刷新的问题
- **日志路径失效**: 修复设置日志文件夹路径后重启丢失的问题

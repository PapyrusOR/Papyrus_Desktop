## 1. 产品概述

### 1.1 产品简介

**Papyrus（莎草纸）** 是一款专注于高强度模型记忆的极简、高效、全键盘驱动的 AI Agent 驱动暗记（SRS）复习引擎。

**核心理念**: "大道至简" - 通过极简交互帮助用户进入深度复习的"心流"状态。

### 1.2 目标用户

- 需要记忆大量知识点的学生和终身学习者
- 追求效率、喜欢键盘操作的高级用户
- 需要无障碍支持的视障/行动不便用户
- 使用 Anki 等工具但希望更简洁体验的用户

### 1.3 核心价值主张

1. **极简交互**: 全流程键盘驱动，无需触碰鼠标
2. **智能调度**: 基于 SM-2 算法的间隔重复
3. **AI 辅助**: 支持多种 AI 提供商的智能学习助手
4. **数据自主**: 本地存储，隐私安全
5. **无障碍优先**: 达到 WCAG 2.1 AAA 级标准

## 4. 用户界面设计

### 4.1 窗口模式支持

本应用支持两种窗口模式，适应不同使用场景：

| 模式 | 尺寸 | 适用场景 |
|------|------|----------|
| **小窗模式** | 800×600 ~ 1024×768 | 边学边记、悬浮复习、多任务并行 |
| **全屏模式** | 最大化/全屏 | 专注复习、深度阅读、长时间使用 |

**模式特性**:
- **小窗模式**: 精简界面元素，保留核心复习功能，支持置顶悬浮
- **全屏模式**: 完整三栏布局（Sidebar + Main + Chat），沉浸式体验
- **自适应布局**: 根据窗口尺寸自动调整界面元素和字体大小
- **快捷键切换**: `F11` 切换全屏，`Ctrl + =/-` 调整缩放

## 5. 数据模型

### 5.1 卡片数据结构

```typescript
interface Card {
  id: string;           // UUID
  q: string;            // 问题（正面）
  a: string;            // 答案（背面）
  next_review: number;  // 下次复习时间戳
  interval: number;     // 间隔（秒）
  ef: number;           // 简易因子
  repetitions: number;  // 连续成功次数
  tags: string[];       // 标签列表
}
```

### 5.2 笔记数据结构

```typescript
interface Note {
  id: string;           // UUID
  title: string;        // 标题
  folder: string;       // 所属文件夹
  content: string;      // Markdown 内容
  preview: string;      // 预览（前100字）
  tags: string[];       // 标签列表
  created_at: number;   // 创建时间戳
  updated_at: number;   // 更新时间戳
  word_count: number;   // 字数统计
}
```

### 5.3 AI 配置数据结构

interface AIConfig {

  max_tokens: number;   // 最大 token 数

## 6. 非功能需求

### 6.1 性能要求

| 指标 | 目标 |
|------|------|
| 首屏加载 | < 3 秒 |
| 卡片切换 | < 100ms |
| AI 响应 | < 5 秒（流式输出） |
| 搜索响应 | < 500ms |
| 内存占用 | < 500MB |

### 6.2 兼容性要求

| 平台 | 支持版本 |
|------|----------|
| Windows | 10/11 |
| macOS | 12+ (Apple Silicon) |
| Linux | Ubuntu 20.04+ |

### 6.3 安全要求

- API Key 本地加密存储
- 不使用外部 CDN
- 支持离线使用
- 定期安全审计

### 6.4 可访问性要求

- 通过 WCAG 2.1 AAA 级测试
- 支持屏幕阅读器
- 支持键盘完全操作
- 高对比度模式

### 6.5 响应式设计要求

| 场景 | 最小尺寸 | 布局适配 |
|------|----------|----------|
| 小窗模式 | 800×600 | 单栏布局，隐藏 Sidebar 和 ChatPanel |
| 标准窗口 | 1280×720 | 双栏布局（Sidebar + Main） |
| 全屏模式 | 1920×1080 | 三栏完整布局 |
| 超宽屏 | 2560×1080+ | 自适应边距，内容居中 |

**断点规则**:
- `< 1024px`: 进入紧凑模式，侧边栏可折叠
- `1024px ~ 1440px`: 标准模式，可选显示 ChatPanel
- `> 1440px`: 完整模式，三栏布局

**字体缩放**: 支持 100% ~ 150% 字体缩放，适应不同屏幕 DPI

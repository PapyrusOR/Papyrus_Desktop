# 无障碍设置说明

Papyrus 提供了一系列无障碍选项，帮助不同需求的用户更舒适地使用应用。

## 访问方式

设置 → 无障碍

## 可用选项

### 1. 减少动画
**功能**：减弱界面动画效果  
**适用人群**：对动画敏感、容易晕动症的用户  
**效果**：
- 禁用过渡动画
- 减少悬停效果
- 简化页面切换

### 2. 高对比度
**功能**：增强文字与背景的对比度  
**适用人群**：视力不佳、需要更清晰界面的用户  
**效果**：
- 提高文字对比度至 AAA 级标准
- 增强边框和分隔线
- 优化焦点指示器

### 3. 屏幕阅读器优化
**功能**：优化界面元素的屏幕阅读器体验  
**适用人群**：使用屏幕阅读器的视障用户  
**效果**：
- 增强 ARIA 标签
- 优化朗读顺序
- 提供更多上下文信息

### 4. 焦点指示器
**功能**：始终显示键盘焦点高亮  
**适用人群**：仅使用键盘导航的用户  
**效果**：
- 清晰的焦点轮廓
- 高对比度的焦点颜色
- 易于追踪当前位置

### 5. 大光标
**功能**：放大鼠标指针  
**适用人群**：视力不佳、难以找到光标的用户  
**效果**：
- 增大光标尺寸
- 提高光标可见度

## 字体大小调整

字体大小设置位于：**设置 → 外观 → 字体大小**

可选大小：
- **小**：紧凑界面，适合小屏幕
- **中**：默认大小
- **大**：更大的文字，提高可读性

## 键盘快捷键

无障碍相关的快捷键：

| 快捷键 | 功能 |
|--------|------|
| `Tab` | 在可交互元素间导航 |
| `Shift + Tab` | 反向导航 |
| `Enter` / `Space` | 激活按钮或链接 |
| `Escape` | 关闭弹窗或菜单 |
| `Ctrl + K` | 打开搜索 |

## 屏幕阅读器支持

Papyrus 支持以下屏幕阅读器：

- **Windows**：NVDA、JAWS、讲述人
- **macOS**：VoiceOver
- **Linux**：Orca

## ⌨️ 键盘导航测试（AA 级）

### 基础导航

| 测试项 | 操作 | 期望结果 |
|--------|------|----------|
| Skip Link | 按 `Tab` 在页面加载后 | 显示"跳转到主内容"链接 |
| 侧边栏导航 | 按 `Tab` 遍历侧边栏按钮 | 每个按钮都可聚焦 |
| 页面切换 | `Tab` 到侧边栏按钮，按 `Enter` | 切换到对应页面 |
| 焦点可见 | 观察焦点样式 | 蓝色边框 + 偏移清晰可见 |
| 焦点顺序 | 按 `Tab` 多次 | 顺序符合视觉顺序 |

### 测试步骤

1. **断开鼠标**，仅用键盘操作
2. 按 `Tab` 遍历所有可交互元素
3. 确认每个按钮/链接可用 `Enter` 或 `Space` 激活
4. 确认模态框可用 `Escape` 关闭

## 🎨 对比度测试（AA/AAA 级）

### 工具

- [WebAIM Contrast Checker](https://webaim.org/resources/contrastchecker/)
- Chrome DevTools → Elements → 颜色值旁的对比度提示

### 测试项

| 元素 | AA 标准 | AAA 标准 | 测试方法 |
|------|---------|----------|----------|
| 正文文本 | ≥ 4.5:1 | ≥ 7:1 | 使用取色器检查 |
| 大文本 (≥18pt 粗体) | ≥ 3:1 | ≥ 4.5:1 | 使用取色器检查 |
| 焦点指示器 | ≥ 3:1 | ≥ 4.5:1 | 检查焦点样式 |

### 快速检查

```javascript
// 在控制台运行，检查所有文本元素对比度
// 注意：需要安装 axe-core
axe.run().then(results => {
  console.log(results.violations.filter(v => v.id.includes('contrast')));
});
```

## 🔊 屏幕阅读器测试（AA 级）

### Windows - NVDA

1. 下载安装 [NVDA](https://www.nvaccess.org/download/)
2. 打开 Papyrus 应用
3. 使用 `Insert + ↓` 开始朗读
4. 使用 `Tab` 导航

1. 启用 VoiceOver：`Cmd + F5`

3. 使用 `Ctrl + Option + Space` 激活

## 🔧 常见问题排查

### 问题：焦点样式不显示

**检查**：
```css
/* 确保没有覆盖 */
:focus-visible {
  outline: 2px solid #206CCF !important;
}
```

### 问题：屏幕阅读器不朗读更新

**检查**：
- `aria-live` 区域是否正确设置
- 消息是否在 `ScreenReaderAnnouncer` 中发送

### 问题：设置不持久化

**检查**：
- localStorage 是否可用
- 浏览器隐私模式是否阻止存储

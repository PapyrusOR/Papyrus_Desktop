# Papyrus 更新日志

## v1.2.0-beta (2026-03-09)

### 🎉 重大更新

#### 核心算法升级
- **SM-2 间隔重复算法**：替换原有简单算法，提供科学的复习间隔计算
  - 根据答题质量动态调整复习间隔
  - 每张卡片独立的难度系数 (EF)
  - 自动追踪连续正确次数
  - 完全向后兼容旧数据

#### AI 助手集成
- **全新 AI 侧边栏**：深色主题，现代化设计
  - 纯对话模式，自然语言交互
  - 自动感知当前学习卡片上下文
  - 支持多轮对话，保留历史记忆
  - 实时状态指示

- **多模型支持**：
  - OpenAI (GPT-4, GPT-3.5-turbo)
  - Anthropic (Claude 3 系列)
  - Ollama (本地模型)
  - 自定义 API 端点

- **完整的模型管理**：
  - GUI 界面添加/编辑/删除模型
  - 从 API 自动刷新模型列表
  - 多提供商快速切换
  - 参数精细调整 (Temperature, Max Tokens)

- **AI 工具调用**：
  - 创建/更新/删除卡片
  - 搜索知识库
  - 学习统计分析
  - 操作审批机制，确保数据安全

#### 用户体验优化
- **隐私保护**：API Key 在设置界面中隐藏显示
- **智能备份**：每小时自动备份，防止数据丢失
- **依赖检测**：启动时自动检查依赖，友好提示
- **错误处理**：完善的异常捕获和用户提示

### 📝 详细改进

#### 学习算法
- 实现标准 SM-2 算法
- 三级评分映射到 SM-2 质量评分
- 动态调整难度系数 (1.3 - 无上限)
- 首次复习：1天 → 6天 → 按 EF 递增

#### AI 功能
- 智能提示（不透露答案）
- 答案深度解释
- 生成相关练习题
- 错题分析
- 学习统计

#### 设置系统
- 三标签页设计：模型管理 / API配置 / 参数设置
- 支持多个 AI 提供商配置
- 实时参数调整
- 配置持久化存储

#### 技术架构
- 模块化设计，AI 功能完全独立
- 安全隔离，缺少依赖时主程序正常运行
- 异步处理，UI 不卡顿
- 完整的错误日志

### 🔧 技术细节

#### 新增文件
```
src/ai/
├── __init__.py          # AI 模块初始化
├── config.py            # 配置管理
├── provider.py          # AI 提供商接口

```

#### 依赖要求
- Python 3.7+
- tkinter (内置)
- requests (新增，可选)

#### 数据格式变更
卡片数据新增字段（向后兼容）：
```json
{
  "q": "题目",
  "a": "答案",
  "next_review": 0,
  "interval": 0,
  "ef": 2.5,           // 新增：难度系数
  "repetitions": 0     // 新增：连续正确次数
}
```

### 📚 使用指南

#### 安装 AI 功能
```bash
pip install requests
```

#### 配置 AI
1. 启动程序，点击右侧 AI 助手的 "⚙" 按钮
2. 在 "API配置" 标签页输入 API Key
3. 在 "模型管理" 标签页选择或添加模型
4. 保存设置

#### 使用 AI 对话
- 直接输入问题，AI 会自动理解当前卡片
- 例如："帮我解释这道题"
- 例如："创建一张关于递归的卡片"
- 例如："搜索所有 Python 相关的题"

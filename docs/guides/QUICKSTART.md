# Papyrus v1.2.0-beta 快速启动指南

## 🚀 5分钟上手

### 步骤 1：安装依赖（可选）

如果你想使用 AI 功能：
```bash
pip install requests
```

如果只使用基础学习功能，可以跳过此步骤。

### 步骤 2：启动程序

```bash
python src/Papyrus.pyw
```

### 步骤 3：开始学习

#### 添加卡片
1. 点击菜单 "操作" → "添加新卷轴"
2. 输入题目和答案
3. 点击保存

#### 批量导入
1. 准备 TXT 文件，格式：`题目===答案`
2. 点击菜单 "操作" → "批量导入 (TXT)"
3. 选择文件

#### 复习卡片
- **Space**：显示答案
- **1**：忘记（1天后复习）
- **2**：模糊（正常间隔）
- **3**：秒杀（快速增长）

## 🤖 AI 功能配置

### 方式 1：使用 OpenAI

1. 获取 API Key：https://platform.openai.com/api-keys
2. 点击 AI 助手的 "⚙" 按钮
3. 在 "API配置" → "OPENAI" 中输入 API Key
4. 在 "模型管理" 中选择模型（如 gpt-3.5-turbo）
5. 保存设置

### 方式 2：使用 Ollama（免费本地）

1. 安装 Ollama：https://ollama.ai
2. 拉取模型：
   ```bash
   ollama pull qwen:0.5b  # 小模型，快速
   # 或
   ollama pull llama2     # 更强大
   ```
3. 在 Papyrus 中：
   - 提供商选择：ollama
   - 模型管理：添加模型名称（如 qwen:0.5b）
4. 保存设置

### 方式 3：使用自定义 API

如果你有其他兼容 OpenAI API 的服务：
1. 提供商选择：custom
2. 输入 API Key 和 Base URL
3. 添加模型名称
4. 保存设置

## ⚙️ 常用设置

### 调整 AI 参数

在设置 → 参数标签页：

- **Temperature**（创造性）
  - 0.3：精确、保守
  - 0.7：平衡（推荐）
  - 1.2：创意、发散

- **Max Tokens**（回复长度）
  - 500：简短回答
  - 2000：详细解释（推荐）
  - 4000：长篇内容

### 管理模型

在设置 → 模型管理标签页：

- **➕ 添加**：手动添加模型名称
- **✏️ 编辑**：修改模型名称
- **🗑️ 删除**：删除不需要的模型
- **🔄 从API刷新**：自动获取可用模型列表

## 🔧 故障排除

### AI 功能不可用

**问题**：右侧显示占位面板

**解决**：
1. 检查是否安装 requests：`pip list | grep requests`
2. 重启程序
3. 查看控制台错误信息

### API 调用失败

**问题**：显示 "错误: API调用失败"

**解决**：
1. 检查 API Key 是否正确
2. 检查网络连接
3. 确认 API 额度是否充足
4. 尝试切换模型

### Ollama 连接失败

**问题**："Ollama调用失败"

**解决**：
1. 确认 Ollama 服务运行：`ollama serve`
2. 检查端口：默认 11434
3. 确认模型已下载：`ollama list`

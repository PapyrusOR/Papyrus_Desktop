# 🎛️ AI输入框功能说明

## 功能概览

AI助手输入框包含以下功能：

### 1. 模式选择 (Mode Selection)

**✨ Agent 模式**
- 启用AI工具调用功能
- AI可以执行操作（创建卡片、搜索、统计等）
- 适合需要AI主动帮助的场景
- 按钮显示：蓝色高亮

**💬 Chat 模式**
- 禁用工具调用
- 纯对话模式，AI只回答问题
- 适合简单咨询和讨论
- 按钮显示：灰色

**切换方式：**
点击对应的模式按钮即可切换

---

### 2. 模型选择 (Model Selection)

**功能：**
- 快速切换不同的AI模型
- 下拉菜单显示当前提供商的所有可用模型
- 选择后立即生效并保存配置

**显示位置：**
输入框上方右侧，显示为下拉菜单

**支持的模型：**
- 根据配置文件中的模型列表动态显示
- 可在设置中添加/编辑/删除模型

## 使用示例

### 场景1：使用Agent模式创建卡片

1. 确保 **✨ Agent** 按钮是蓝色（已选中）
2. 输入："帮我创建一张关于Python装饰器的卡片"
3. 按Enter发送
4. AI会调用工具创建卡片

### 场景2：使用Chat模式咨询

1. 点击 **💬 Chat** 按钮切换到Chat模式
2. 输入："什么是装饰器？"
3. 按Enter发送
4. AI只会回答问题，不会执行操作

### 场景3：切换模型

1. 点击右上角的模型下拉菜单
2. 选择想要使用的模型（如 gpt-4, claude-3等）
3. 模型立即切换，下次对话使用新模型

## 技术实现

### 模式切换
```python
self.agent_mode = tk.BooleanVar(value=True)

# Agent模式：启用工具
if self.agent_mode.get() and self.card_tools:
    system_prompt += tools_definition

# Chat模式：禁用工具
else:
    # 不添加工具定义
```

### 模型切换
```python
def on_model_change(self, model):
    self.ai_manager.config.config["current_model"] = model
    self.ai_manager.config.save_config()
```

## 注意事项

1. **模式选择**：Agent模式会消耗更多token，因为包含工具定义
2. **模型选择**：不同模型的能力和价格不同，请根据需求选择
3. **快捷键**：使用Enter快速发送，Shift+Enter换行
